# ease-chip-remove

Animated removable chip/tag component for EaseMotion CSS.

## Features

- Bounce entrance animation
- Shrink-and-fade removal
- Slide-out removal variant
- Color-coded chip styles
- Interactive close button
- Hover elevation effect
- Responsive flex layout
- Pure CSS animations
